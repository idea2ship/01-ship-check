> 캐러셀이 "정리된 설명서"라면 릴스는 "5초 안에 마음 사로잡기".
> 사이즈: **1080×1920 (9:16)**, 길이: **15~30초** 권장.

## 컨셉

3~5초 hook → 화면 녹화로 동작 보여주기 → 결과 강조 → CTA.

**핵심 원칙**: 음성 없이도 이해 가능하게. 알고리즘은 무음 재생도 많이 돌리기 때문. 자막 + 텍스트 오버레이 필수.

## 촬영/제작 팁

### 화면 녹화 (Demo 부분)

- **macOS**: QuickTime Player → File → New Screen Recording → 사이트 영역만 선택
- 화면 캡처 후 **CleanShot X** 또는 **ScreenStudio**로 zoom/highlight 효과 추가
- 결과 카드 등장 부분은 **0.7배속 슬로우 모션**으로 강조하면 좋음

### 자막 / 오버레이

- **CapCut**으로 자막 + 모션 (무료, 직관적)
- 폰트: Pretendard Bold (한글) / Inter Black (영문)
- 자막 색: 흰색 + 검정 stroke (또는 mint accent)
- 자막 위치: 화면 하단 1/3 (safe area)

### 음악

- 무료 BGM: Pixabay Music, Mixkit, YouTube Audio Library
- 추천 톤: lo-fi, future garage, indie electronic (calm + optimistic — 우리 브랜드 무드)
- 비트가 강한 곡은 피하기 (cozy maker 분위기 유지)

### 첫 1초 (썸네일)

- 인스타 릴스의 첫 프레임이 곧 썸네일
- "/1>" 모노그램 + 큰 텍스트 hook이 들어가도록 첫 컷 디자인

## 빠른 시작 — 5분 안에 만드는 첫 릴스

1. QuickTime으로 사이트 입력 → 결과 카드까지 한 번에 화면 녹화 (15초)
2. CapCut import
3. 시작 3초에 "/1> Ship Check — 1주일 안에 만들 수 있나?" 텍스트 오버레이
4. 결과 카드 등장 부분 zoom + slow motion
5. 마지막 3초에 "idea2ship.xyz" 텍스트 + fade-out
6. 적당한 lo-fi BGM 깔기
7. Export → Instagram 업로드

→ 캐러셀 게시 직전/직후 같이 올리면 알고리즘 노출 ↑
